# たてしょうぎ (Tate-Shogi)

3列×9段のコンパクトな盤面で、王（玉）・金・歩のみを使用して遊ぶミニ将棋（変則将棋）アプリです。  
HTML/CSS/JavaScriptの単一ファイルで構成されており、ブラウザですぐにプレイできます。対人戦（2人交互プレイ）および簡易的なAI（CPU）との対戦機能を搭載しています。

## 🚀 特徴

- **シンプルなルールと短い対局時間**: 通常の将棋よりも大幅に少ない駒（王・金・歩）とコンパクトな盤面により、初心者でも短時間でエッセンスを楽しめます。
- **2つの対戦モード**: 
  - **対人で遊ぶ**: 1台の端末を交互に操作して2人で対戦。
  - **CPUと対戦する**: AIとの対戦モード。プレイヤーの先手・後手を選択可能です。
- **本格的な将棋ルールを再現**: 「二歩」「打ち歩詰め」「行き所のない歩の禁止」などの本格的な禁じ手・ルール判定や、「と金」への成りを正確に実装しています。
- **美しい和風デザイン**: モバイル・デスクトップ両対応のレスポンシブな盤面と、上品な木目調・和風カラーパレットを採用しています。

## 🎯 ルール概要

通常の将棋のルールをベースに、以下の構成・ルールで進行します。

### 1. 初期配置と盤面
- **盤面**: 3列 × 9段 (計27マス)
- **駒の構成（各プレイヤー）**:
  - 王将（玉将）× 1
  - 金将 × 2
  - 歩兵 × 3
- **初期配置**:
  - 1段目（後手）/ 9段目（先手）：[ 金, 王/玉, 金 ]
  - 2段目（後手）/ 8段目（先手）：[ 歩, 歩, 歩 ]

### 2. 駒の動き
- **王 / 玉**: 周囲8マスのいずれかに1マス動ける。
- **金**: 前・後・左・右・斜め前の計6方向に1マス動ける（斜め後ろには動けない）。
- **歩**: まっすぐ前に1マスだけ動ける。
- **と金（成った歩）**: 金と同じ動きになる。

### 3. 成り（プロモーション）
- 歩が敵陣（自分から見て奥の3段）に入った際、「と金」に成るかどうかを選択できます。
- 一番奥の段に進む場合は、動けなくなるのを防ぐため**強制的に成ります**。

### 4. 持ち駒・打つ
- 相手の駒を取ると自分の持ち駒になり、盤上の空きマスに消費して配置（打つ）できます。
- **禁じ手（合法手チェック）**:
  - **二歩**: 同じ列に自分の成っていない「歩」がすでにある場合、その列に新たな歩を打つことはできません。
  - **打ち歩詰め**: 持ち駒の歩を打って相手の玉を即座に詰ませることはできません（盤上の歩で詰ませる「突き歩詰め」は有効）。
  - **行き所のない駒**: 歩を一番奥の段に打つことはできません。

### 5. 勝敗の決定
- 相手の玉を完全に動けなくする（**詰み**）ことで勝利となります。
- また、自分の手番で動かせる合法手が1つも存在しない場合も敗北となります。

## 💻 技術スタックと仕組み

- **フロントエンド**: HTML5, CSS3 (変数を用いたモダンなカラー設計、Grid/Flexboxによるレイアウト), Vanilla JavaScript
- **思考エンジン (CPU)**:
  - 盤面が27マスと軽量なため、JavaScriptによる**ミニマックス法（ネガマックス法）**および**αβプライニング（Alpha-Beta Pruning）**を用いて探索（深さ3手）を行っています。
  - 駒の価値評価（金=5, と金=6, 歩=1, 持ち駒への微小ボーナス）をベースに、王手の回避・付与度合いを加味した評価関数をリアルタイムに計算します。
  - 最善手近傍の選択肢からランダムに選ぶことで、CPUの挙動に多様性を持たせています。

## 🛠️ 導入・実行方法

外部ライブラリやビルド環境は一切不要です。

1. このリポジトリをクローンまたはファイルをダウンロードします。
   ```bash
   git clone https://github.com/YOUR-USERNAME/tate-shogi.git
   ```
2. `index.html`（または該当のHTMLファイル）をブラウザでダブルクリックして開くだけでプレイ可能です。
