# merge_folders

複数のフォルダに散らばったファイルを、1つの新しいフォルダにまとめてコピーするWindows用ツールです。

すべてマウス操作（ダイアログ選択）だけで完結し、コマンドやパスを手入力する必要はありません。

**[▶ index.ps1 をダウンロード](https://tosnkj.github.io/merge_folders/index.ps1)**

## できること

- 複数のフォルダを選んで、その中のファイルを1つのフォルダにまとめてコピー
- フォルダの一覧からShiftキー（範囲選択）・Ctrlキー（個別選択）で複数選択可能
- 同じ名前のファイルがあった場合は自動的にリネーム（`ファイル名_1.拡張子` のように連番を付与）して、上書きを防止
- サブフォルダの中身も含めるかどうかを選択可能
- 元のファイルはコピーされるだけで、移動・削除はされません（安全）

## 必要な環境

- Windows 10 / 11
- PowerShell（Windowsに標準搭載されているもので動作します。追加インストール不要）

## ダウンロード方法

1. このページの [`index.ps1`](./index.ps1) を開く
2. 右上あたりにある「Raw」ボタンをクリック
3. 表示されたページで `Ctrl + S` を押して、任意の場所に保存(ファイル名は `index.ps1` のままにしてください)

## 使い方

1. 保存した `index.ps1` を右クリック →「PowerShellで実行」を選択
2. 実行ポリシーに関する警告が出て動かない場合は、下記の「実行できない場合」を参照
3. 画面の指示に従って進めます:
   1. まとめたいフォルダが入っている「親フォルダ」を選択
   2. 表示された一覧から、まとめたいフォルダをShift/Ctrlキーで複数選択
   3. 別の場所からも追加したい場合は「はい」を選んで1〜2を繰り返し、終わったら「いいえ」
   4. まとめ先の新しいフォルダを選択（その場で新規作成も可能）
   5. サブフォルダの中身も含めるか選択
4. 自動でコピーが実行され、完了するとまとめ先フォルダが自動で開きます

## 実行できない場合

Windowsのセキュリティ設定により、ダウンロードしたスクリプトがそのままでは実行できないことがあります。その場合は以下の手順をお試しください。

1. PowerShellを開く(スタートメニューで「PowerShell」と検索)
2. 保存した `index.ps1` があるフォルダに移動:
   ```powershell
   cd "保存したフォルダのパス"
   ```
3. 実行ポリシーを一時的に緩和:
   ```powershell
   Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
   ```
   (この設定は、今開いているPowerShellウィンドウを閉じるまでの間だけ有効です。パソコン全体の設定は変更されません)
4. スクリプトを実行:
   ```powershell
   .\index.ps1
   ```

## 安全性について

- このツールはローカルのファイルをコピーするだけで、外部への通信は一切行いません
- インターネット接続も不要です
- ソースコードはこのリポジトリ内の `index.ps1` に全て記載されているため、内容を確認してから実行いただけます
